# GitHub User Finder

Автор: Канищев Максим

## Описание
Это графическое приложение для поиска пользователей GitHub по имени, отображения информации и сохранения избранных в локальный JSON-файл.

## Как использовать API
- Введите имя пользователя и нажмите "Найти".
- Если пользователь найден, его данные появятся на экране.
- Для добавления в избранное нажмите соответствующую кнопку.

## Инструкция по запуску
1. Установите Python 3.
2. Установите необходимые библиотеки: requests.
3. Запустите файл `main.py`.
4. При первом запуске будет создан файл `users.json` для хранения избранных.

## Примеры поиска
- `torvalds`
- `octocat`
